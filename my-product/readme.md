# 女友人像相机 — 产品文档

> 帮不会拍照的男生，在给对象拍照时自动调节对光、曝光等参数，拍出好看的照片。

## 产品定位

**一句话**：给不会拍照的男生用的女友人像相机——自动对光、自动曝光，按快门就行。

**目标用户**：18–35 岁、手机摄影小白的恋爱/约会男性。

**核心差异化**：「给对象拍」专用 + 拍摄中实时曝光/对光优化 + 零学习成本 UI。

## 市场结论（摘要）

- **能力分散存在**：系统相机、一相机、男友照相机、AI Relight 等各自覆盖一部分
- **空白地带**：专注「男友给对象拍 + 实时曝光对光 + 极简交互」的独立产品几乎没有
- **最大风险**：系统相机计算摄影追赶、第三方 App 硬件权限受限、「好看」主观性强

## 下一步

1. 用 MVP 技术方案搭建 Phase 1 原型（人脸优先测光 + 光线检测 + 女友模式）

3. 根据内测反馈决定是否加入 P1 功能（构图辅助、连拍选片）
